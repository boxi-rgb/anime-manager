# Product Definition - anime-manager

## Description
アニメの視聴リストを管理し、各種配信サイトやWikiへのリンクを提供するウェブアプリケーション。

## Problem Statement
ブラウザのクリーンアップ等で視聴データ（Local Storage）が消えてしまうことを防ぎ、永続的に管理したい。

## Target Users
自分自身（個人用ツールとしての運用）。

## Key Goals
- Googleログイン連携によるシームレスなデータ永続化（Google Drive保存）
- AnnictやWiki等の外部APIからの情報自動取得と集約
